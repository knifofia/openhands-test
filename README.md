# YouTube Channel Website

This is a responsive website template for a YouTube channel. It includes sections for showcasing videos, about information, and contact details. The website is designed to be modern, user-friendly, and optimized for all devices.

## Features

- Responsive design that works on mobile, tablet, and desktop
- Featured videos section to highlight your best content
- About section to tell your story
- Contact form for viewers to reach out
- Social media integration
- Smooth scrolling navigation
- Modern and clean design with YouTube-inspired red color scheme

## How to Use

1. Clone this repository
2. Customize the content in `index.html` to match your channel
3. Replace placeholder images with your own thumbnails and branding
4. Update social media links and contact information
5. Modify the styling in `styles.css` if desired
6. Deploy to your preferred hosting service

## Running Locally

To run the website locally, you can use the included Python server:

```bash
python server.py
```

Then open your browser and navigate to http://localhost:53909

## Customization

### To customize the website for your channel:

1. **Update Channel Name**: Replace "My YouTube Channel" with your actual channel name throughout the HTML
2. **Add Your Videos**: Replace the placeholder video cards with your actual videos
3. **Update About Section**: Fill in your channel's story, mission, and statistics
4. **Social Media Links**: Add your actual social media URLs
5. **Contact Information**: Update with your real email address
6. **Colors**: Modify the CSS variables in the `:root` selector to match your branding
